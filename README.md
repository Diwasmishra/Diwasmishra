# 💫 About Me:
I'm a Full-Stack Software Engineer who enjoys building production-ready applications and understanding the infrastructure behind them.<br><br>
Currently, I develop and maintain a production-grade School ERP platform serving 10,000+ students, using React (Vite), Node.js, Express.js, and Microsoft SQL Server. My work involves designing RESTful APIs, optimizing SQL queries and stored procedures, debugging production issues, and delivering features used by thousands of users every day.<br><br>My journey also includes enterprise infrastructure training at Tata Consultancy Services (TCS), where I gained hands-on experience with Windows Server Administration. I worked with Active Directory Domain Services (AD DS), Domain Controllers, Organizational Units (OU), Group Policy (GPO), DNS, DHCP, DFS/DFSR, FSMO Roles, IIS, PowerShell, Windows Admin Center, user and group management, authentication, permissions, file services, storage management, and basic networking concepts. This experience gave me a deeper understanding of how enterprise applications are deployed, managed, secured, and maintained in real-world environments.<br><br>Beyond full-stack development, I'm interested in System Design, Cloud Infrastructure, Machine Learning (NLP) and Blockchain. I enjoy learning technologies from the ground up and applying them to solve practical problems.
