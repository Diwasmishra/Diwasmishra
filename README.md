# 💫 About Me:
I build full stack applications that real people depend on — not demo apps, production systems with thousands of daily users.<br><br>As a Full Stack Engineer, I've spent the last two years working across the entire stack: designing REST APIs in Node.js and Express, building responsive interfaces in React, and optimizing MSSQL databases that need to hold up under enterprise-scale load.<br><br>At DPS Technologies, I led the migration of a legacy VB6-based School ERP into a modern, web-based platform — now used by 30+ schools and 2,000+ students, with 1,000+ daily active users.
